** memory_overview
*** ram内存使用情况
+ 总内存使用了148K， 还有差不多15k的剩余内存
+ 各种解码格式是共用内存的，因此使用的最大内存按照最大格式计算就好 53，788
  + acc 53,788
  + flac 40,520
  + wav 19,148t
  + mp3 34,615
  + wma 49,188
  + amr enc  48,224
  + sinv 43,668
+ 系统使用内存中主要部分
  + 系统heap 32K, 在系统heak中有10K左右的WIFI驱动buffer
  + 网络buffer  16K
  + 任务堆栈 14K
  + WIFI驱动 10K
+ 进一步压缩的话，应该可以再提供20K左右内存，算法使用的内存可以最多到75K左右
*** nor 使用情况
+ 目前代码占用的nor空间为 1,364,884
+ 代码大小 559,612 主要有几部分
  + 解码库 188K
  + wifi驱动 138K
  + 网络协议栈  50K
  + 应用框架  54k
+ rodata占用空间793,916  主要由几部分：
  + wifi模组的firmware代码 171,856
  + 提示音mp3文件 458,652
  + 各种解码格式的码表 97,719
  + 其他部分
